# craft-kit

Un solo plugin (`craft`) con cuatro skills complementarias que no se pisan:

| Skill | Trabajo |
|---|---|
| frontend-design | Genera la UI con buen gusto (evita el look generico de IA) |
| impeccable | Audita la UI ya hecha y marca anti-patterns concretos |
| design-motion-principles | Disena animaciones y movimiento |
| stop-slop | Limpia los textos y microcopy de tells de IA |

Cada una ocupa un trabajo distinto: una construye, otra verifica, otra anima, otra escribe.

## Instalar
```
/plugin marketplace add <usuario>/craft-kit
/plugin install craft@craft-kit
```
Pensado para dejarlo prendido siempre.
